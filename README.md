# X Silent Blocker
- Chrome拡張機能パッケージ（Chrome拡張機能が使用できるブラウザであれば動作可能です。作者はHelium Browserで使用してます）
- X(旧ツイッター)で指定したキーワードやユーザーIDを指定して非表示にします
- ローカル環境で実行されるため、X標準のブロック機能とは違って相手に気づかれずに自衛できます

## Note
- リポスト先のアカウントをサイレントブロックする場合、その投稿のページに一度移動してから実行する必要があります。

## Install
1. このリポジトリをクローンまたはZIPダウンロード
2. Chromeの場合 chrome://extensions/ を開く
3. 「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」→ DLしたフォルダを選択

## How to use
### キーワードでブロックする場合
1. 拡張機能のオプションから設定画面に飛ぶ
2. キーワードフィルターで非表示にするキーワードを追加

### ユーザーIDでブロックする場合
- 「もっと見る（三点タブ）」からブロックする場合
「(ユーザーID)さんをサイレントブロック」をクリックする
- またはオプション下のサイレントブロック項目で手動でIDを入力

## 設定のインポート・エクスポート
エクスポートボタンでjsonファイルをエクスポート、インポートボタンでjsonを選択して設定を復元できます
