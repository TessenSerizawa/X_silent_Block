# X Silent Blocker
- X(旧ツイッター)で指定したキーワードやユーザーIDを指定して非表示にします
- ローカル環境で実行されるため、X標準のブロック機能とは違って相手に気づかれずに自衛できます

## Install
1. DownloadZipからダウンロード
2. 解凍したフォルダをデベロッパーモードをオンにした「拡張機能」ページの「パッケージ化されていない拡張機能を読み込む」かフォルダごと選択

## How to use
### キーワードでブロックする場合
1. 拡張機能のオプションから設定画面に飛ぶ
2. キーワードフィルターで非表示にするキーワードを追加

### ユーザーIDでブロックする場合
- 「もっと見る（三点タブ）」からブロックする場合
「(ユーザーID)さんをサイレントブロック」をクリックする
